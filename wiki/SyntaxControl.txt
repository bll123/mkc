### Syntax: Control Statements

##### if Statements

    if (<expr>) {
      <statement-list>
    }

    if (<expr>) {
      <statement-list>
    } else {
      <statement-list>
    }

    if (<expr>) {
      <statement-list>
    } else if (<expr>) {
      <statement-list>
    }

    if (<expr>) {
      <statement-list>
    } else if (<expr>) {
      <statement-list>
    } else if (<expr>) {
      <statement-list>
    } else {
      <statement-list>
    }

##### exit statement

Stops processing.  Used when debugging a .mkc configuration file.
All statements after the __exit__ statement are ignored.

    exit;

##### while statement

    while (<expr>) {
      <statement-list>
    }

__break__ and __continue__ statements may appear in while loops.

    break;
    continue;

##### foreach statement (not yet implemented)

    foreach <var-name> <list> {
      <statement-list>
    }

    foreach <var-name> <integer-value-1>..<integer-value-2> {
      <statement-list>
    }
