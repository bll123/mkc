### Building mkc

##### Cleaning

`make realclean` will clean up everything except for the files
generated by _bison_ and _flex_.

`make distclean` will clean everything.

##### Requisites

The files generated by _bison_ and _flex_ are contained in the source
distribution. _bison_ and _flex_ are not required unless a `make
distclean` is done.

* C compiler
* pkg-config
* libpcre2-8
* lorder
  For future use. Recommended.
  Linux doesn't ship this, an lorder script is included for Linux.
* rm, cp, etc.
* bison
* flex

##### Building

The bootstrap process will build _mkc_ three times to create a
functioning _mkc_ program.  This process is optimized to only
re-compile what is necessary.

`make`

##### Testing

`./tests/runtests.sh`
