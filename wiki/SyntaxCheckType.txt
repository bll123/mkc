### check_type

Checks if _type-name_ exists.

    check_type <type-name>;
    check_type <type-name> { <attribute-statements> }

Generated Name: Prefix __\_type\___
<br>Example: \_type\_uid\_t

##### Examples

    check_type int64_t;
    check_type uid_t {
      header unistd.h;
    }
