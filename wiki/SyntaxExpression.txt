### Syntax: Expressions

Precedence of operators is the same as in the C language.

(sourceforge's version of markdown is eating the &plus; symbol within
code blocks).

Binary Operations on Integers

    plus - * / % && || == != < <= > >=

Binary Operations on Strings

    eq ne lt le gt ge

Unary Operations on Integers

    plus - !

Examples

    set a 1 plus 2;
    set b 0;
    set c ${a} && ${b};
    if (${a} || ${b}) {
       print 'hello';
    }
