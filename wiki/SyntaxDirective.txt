### Syntax: Directives

##### Internal Debugging

    mkcdebug printconfig;           # prints mkc configuration
    mkcdebug printinternal;         # prints internal values
    mkcdebug printprof;             # prints profile names
    mkcdebug printprofvar <profile>;    # user defined profile
    mkcdebug printprofvar internal;
    mkcdebug printprofvar internalall;  # including temporary variables
    mkcdebug printprofvar test;         # most variables
