### check\_constant

    check_constant <constant-name>;
    check_constant <constant-name> { <attribute-statements> }

Checks if _constant-name_ exists and is numeric.

Generated Name: Prefix __\_constant\___
<br>Example: \_constant\_EPERM

##### Examples

    check_constant EPERM {
      header errno.h;
    }
