### Check Define

    check_define <macro-name>;
    check_define <macro-name> { <attribute-statements> }

Checks if _macro-name_ exists as a macro.

Generated Name: Prefix __\_define\___
<br>Example: _define_EPERM

##### Examples

    check_define MAX {
      header sys/param.h;
    }
