### Check Size

    check_size <type-name>;
    check_size <type-name> { <attribute-statements> }

Checks the size of _type-name_.  The check-name will be set to the
size of the type or zero if the type does not exist.

Generated Name: Prefix __\_size\___
<br>Example: _size_int64_t

##### Examples

    check_size int64_t {
      header stdint.h;
    }
