### Check Function

    check_function <function-name>;
    check_function <function-name> { <attribute-statements> }

Checks if _function-name_ exists.  The function is determined to exist
if there is a macro defined with that name, or if the function is
found.

Generated Name: Prefix __\_function\___
<br>Example: _function_statvfs

##### Examples

    check_function malloc;
    check_function fstat {
      header sys/stat.h;
    }
    if (! MKC_SYS_WINDOWS) {
      check_function MAX {
        header sys/param.h;
        name _function_max;
      }
    }
