#### Requirements

The files generated by bison and flex are shipped with the source.  A
_make distclean_ will remove these files.  bison and flex are not
required if building from the source distribution.

Build Requirements

- C compiler
- libpcre2-8
- lorder
  For future use. Recommended.
  Linux doesn't ship this, an lorder script is included for Linux.
- bison
- flex
- pkg-config

Runtime Requirements

- libpcre2-8

#### Building

The bootstrap process will build _mkc_ three times.

make realclean
make

#### Testing

./tests/runtests.sh

#### Install

PREFIX=target-dir make test-install
